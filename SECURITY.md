# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Movie Fairy, please report it by opening a [GitHub Issue](https://github.com/hugcosmos/MovieFairy/issues).

We will respond as soon as possible and work with you to resolve the issue.

## Security Practices

- **No backend**: This is a purely static web application with no server-side code, no database, and no user data collection.
- **No third-party dependencies**: Built with vanilla HTML, CSS, and JavaScript. No npm packages or external SDKs.
- **Content Security Policy**: Strict CSP headers are enforced via Cloudflare Pages.
- **HTTPS**: Served exclusively over HTTPS via Cloudflare Pages.
- **Open source**: All code is publicly auditable.
